# 太空生存戰
## 目標

避開隕石並在太空裡生存下去，射爆隕石取得分數。
努力打破自己的最高分紀錄吧！

## 控制

1. 用左右鍵或AD鍵操控飛船。
2. 用SPACE發射子彈。

## 寶物

射爆石頭有一定機率掉落寶物。
1. 閃電：從單發子彈變成雙發子彈，持續5秒。
2. 盾牌：恢復左上角生命條的20%生命值。

## 生命

左上角有一條生命條，右上角有三架飛船。<br /> 
撞到越大的隕石，左上角生命條的生命值就會扣越多。<br /> 
當左上角生命條的生命值達到0%時，右上角飛船就會減少一架。<br /> 
當三架飛船都沒有被隕石摧毀了，遊戲便結束。<br />
